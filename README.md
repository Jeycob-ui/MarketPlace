# MarketPlace (Prototype)

Proyecto ejemplo: plataforma básica de compra/venta con Node.js, Express y vistas Pug.

Instalación:

```powershell
cd mi-proyecto
npm install
npm run dev
```

Características:
- Registro/Inicio de sesión (roles: admin, vendor, buyer)
- Crear/editar/eliminar productos (vendedores y admin)
- Control de inventario
- Carrito en sesión y checkout (crea pedidos y decrementa stock)
- Panel de administración (lista de usuarios/productos/pedidos)

Notas:
- Este es un prototipo educativo, no usar en producción sin endurecer seguridad.
